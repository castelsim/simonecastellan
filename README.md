# simonecastellan.com

Landing page personale di Simone Castellan — tecnico del suono e produttore audio.

Pagina minimale in stile "Ask AI": pulsanti che aprono ChatGPT, Claude e Grok con un prompt che fa leggere all'assistente il file [`llms.txt`](./llms.txt) per rispondere alle domande su Simone.

- `index.html` — la pagina (HTML/CSS/JS, zero dipendenze)
- `llms.txt` — profilo machine-readable letto dagli assistenti AI
- `CNAME` — dominio personalizzato

Sito statico servito via GitHub Pages.
